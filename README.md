# Sgit

Git implementation in Scala.
 

## Create


- sgit init ✅

## Local changes

- sgit status ✅
- sgit diff ✅
- sgit add (filename or regexp) ✅
- sgit commit ✅

## Commit history

- sgit log ✅
- sgit log -p ✅
- sgit log —stat ✅

## Branches & tags

- sgit branch ✅
- sgit branch -av ✅
- sgit checkout ✅
- sgit tag ✅

## Merge & rebase

- sgit merge ❌
- sgit rebase ❌
- sgit rebase -i ❌
